## 1. サービス概要

**有田焼カタログ** は、日本を代表する伝統工芸「有田焼」の作品・企業情報を集約したオンラインカタログサービスです。

### 主な機能

| 機能 | 一般ユーザー | 会員 (MEMBER) | 会社ユーザー (COMPANY) | 管理者 (ADMIN) |
|------|:---:|:---:|:---:|:---:|
| 商品一覧の閲覧 | ✅ | ✅ | ✅ | ✅ |
| 会社一覧の閲覧 | ✅ | ✅ | ✅ | ✅ |
| 商品詳細の閲覧 | 有田陶器市期間中のみ | ✅ | ✅ | ✅ |
| 価格情報の閲覧 | ❌ | ✅ | ✅ | ✅ |
| 会社・商品の登録 | ❌ | ❌ | ✅ | ✅ |
| 全機能管理 | ❌ | ❌ | ❌ | ✅ |

### サービスURL

```
http://localhost:3000
```

## 3. 会員登録の手順

会員登録を行うことで、商品詳細ページや価格情報など、限定コンテンツにアクセスできるようになります。

### 手順

#### ステップ 1: 登録ページへのアクセス

1. トップページ右上の **「新規登録」** ボタンをクリックします
2. または、URL `http://localhost:3000/ja/signup` に直接アクセスします

#### ステップ 2: 必要事項の入力

以下の項目を入力してください。

| 項目 | 必須 | 説明 |
|------|:---:|------|
| お名前 | ✅ | フルネームで入力してください |
| メールアドレス | ✅ | 有効なメールアドレスを入力してください |
| 電話番号 | ✅ | ログインに使用します（例: +81-090-1234-5678） |
| パスワード | ✅ | 8文字以上、英大文字・英小文字・数字を含む |

> **注意**: 電話番号はログインIDとして使用されます。正確に入力してください。

#### ステップ 3: 登録ボタンをクリック

入力内容を確認し、**「登録する」** ボタンをクリックします。

#### ステップ 4: メール認証

1. 入力したメールアドレスに確認メールが届きます
2. メール内の **「メールアドレスを確認する」** リンクをクリックします
3. 認証が完了するとログインできるようになります

> **注意**: 確認メールが届かない場合は、迷惑メールフォルダをご確認ください。メールは送信後24時間以内に届きます。

#### ステップ 5: ログイン

メール認証完了後、登録した電話番号とパスワードでログインしてください。

## 4. ログイン方法

### 手順

1. トップページ右上の **「ログイン」** ボタンをクリックします
2. **電話番号** を入力します（例: +81-090-1234-5678）
3. **パスワード** を入力します
4. **「ログイン」** ボタンをクリックします

### ログアウト

1. 右上のアカウントメニューをクリックします
2. **「ログアウト」** を選択します

> **ご注意**: ログインセッションは **30日間** 有効です。公共のPCをご利用の場合は必ずログアウトしてください。

## 5. アカウント管理

ログイン後、アカウントページからプロフィール情報を管理できます。

### アカウントページへのアクセス

右上のアカウントメニューから **「アカウント設定」** を選択します。

### 編集可能な情報

| 項目 | 説明 |
|------|------|
| お名前 | 表示名を変更できます |
| 年齢 | 任意 |
| 性別 | 任意（男性 / 女性 / その他） |
| 住所 | 任意 |
| 職業 | 任意 |

> **注意**: 電話番号・メールアドレスの変更は現在サポートされていません。変更が必要な場合はお問い合わせください。

### 情報の更新手順

1. **「アカウント設定」** ページにアクセスします
2. 変更したい項目を編集します
3. **「保存する」** ボタンをクリックします

## 6. 商品・会社の閲覧方法

### 商品一覧の閲覧

1. トップページのナビゲーションから **「商品一覧」** をクリックします
2. 商品がカード形式で表示されます

**一覧で表示される情報:**
- 商品名
- 会社名
- カテゴリ
- タグ

> **注意**: 価格情報は **会員のみ** 閲覧可能です。未登録の方はログインまたは会員登録をしてください。

### 商品詳細の閲覧

商品カードをクリックすると詳細ページが開きます。

> **アクセス制限**: 商品詳細ページは **会員限定** または **有田陶器市期間中（4月29日〜5月5日）** のみ閲覧可能です。

### 会社一覧の閲覧

1. ナビゲーションから **「会社一覧」** をクリックします
2. 登録された窯元・会社の情報が表示されます

### 会社詳細の閲覧

会社カードをクリックすると、その会社の詳細情報と取り扱い商品が表示されます。

## 8. 有田陶器市期間について

### 有田陶器市とは

有田陶器市は毎年ゴールデンウィーク期間中に開催される、国内最大級の陶器市です。

### 特別アクセス期間

**4月29日（昭和の日）〜 5月5日（こどもの日）** の期間中は、非会員の方でも以下のコンテンツが閲覧可能になります。

- 商品詳細ページ
- 一部の商品情報

> **注意**: 価格情報・会員限定商品は市期間中も会員のみが閲覧できます。

### 期間終了後

有田陶器市期間終了後は、再び会員限定のアクセス制限が適用されます。引き続きご利用いただくには、[会員登録](#3-会員登録の手順)をお願いします。

## 9. 会社ユーザー向け機能

会社ユーザー（COMPANY）として登録された方は、以下の機能をご利用いただけます。

### 9-1. 会社登録

#### 手順

1. ログイン後、右上メニューから **「会社ポータル」** を選択します
2. **「会社を登録する」** ボタンをクリックします
3. 以下の情報を入力します

| 項目 | 必須 | 説明 |
|------|:---:|------|
| 会社名 | ✅ | 正式名称で入力してください |
| スラッグ (URL識別子) | ✅ | 半角英数字・ハイフンのみ（例: arita-kiln） |
| 会社説明 | 任意 | 会社の紹介文 |
| 所在地 | 任意 | 住所・地域情報 |
| 画像 | 任意 | 会社のイメージ画像 |

4. **「登録する」** ボタンをクリックします

> **注意**: 1ユーザーにつき登録できる会社は **1社のみ** です。

### 9-2. 商品登録

#### 手順

1. **「会社ポータル」** から **「商品管理」** を選択します
2. **「商品を追加する」** ボタンをクリックします
3. 以下の情報を入力します

| 項目 | 必須 | 説明 |
|------|:---:|------|
| 商品名 | ✅ | 商品の正式名称 |
| 説明 | 任意 | 商品の詳細説明 |
| 価格 | 任意 | 税込価格（円） |
| カテゴリ | 任意 | 商品カテゴリ |
| タグ | 任意 | 検索用タグ（カンマ区切り） |
| 画像 | 任意 | 商品画像 |
| 公開設定 | ✅ | 公開範囲の設定 |

### 9-3. 商品公開設定

商品の公開範囲を3段階で設定できます。

| 設定値 | 表示 | 説明 |
|--------|------|------|
| **PUBLIC** | 全体公開 | ログインしていないユーザーを含む全員が閲覧可能 |
| **MEMBERS_ONLY** | 会員限定 | ログイン済み会員のみ閲覧可能 |
| **WHITELIST** | ホワイトリスト限定 | 指定した会員のみ閲覧可能 |

### 9-4. ホワイトリスト機能

特定の会員のみに商品を公開したい場合は、ホワイトリスト機能をご利用ください。

#### ホワイトリストへの追加方法

1. 商品詳細の管理画面を開きます
2. **「ホワイトリスト管理」** タブを選択します
3. 追加したい会員の会員IDを入力します
4. **「追加する」** ボタンをクリックします

### 9-5. 会社情報の更新

1. **「会社ポータル」** から **「会社設定」** を選択します
2. 変更したい項目を更新します
3. **「保存する」** ボタンをクリックします

## 10. 言語切替

本サービスは以下の8言語に対応しています。

| 言語コード | 言語 |
|-----------|------|
| `ja` | 日本語 |
| `en` | English（英語） |
| `ko` | 한국어（韓国語） |
| `zh-CN` | 简体中文（中国語簡体字） |
| `zh-TW` | 繁體中文（中国語繁体字） |
| `fr` | Français（フランス語） |
| `de` | Deutsch（ドイツ語） |
| `id` | Bahasa Indonesia（インドネシア語） |

### 言語の切替方法

1. ページ上部のナビゲーションバーにある **言語切替メニュー** をクリックします
2. ご希望の言語を選択します
3. ページが選択した言語で表示されます

> URLの構造: `http://localhost:3000/{言語コード}/...`  
> 例: `http://localhost:3000/ja/` (日本語) / `http://localhost:3000/en/` (英語)

## 11. よくある質問 (FAQ)

### Q1. 会員登録は無料ですか？

**A:** はい、会員登録は完全無料です。登録することで、商品詳細・価格情報・会員限定商品などにアクセスできます。

---

### Q2. 電話番号を入力しなければなりませんか？

**A:** はい、電話番号は **ログインID** として使用されるため必須です。入力した電話番号は他のユーザーには公開されません。

---

### Q3. パスワードを忘れてしまいました。どうすればいいですか？

**A:** 現在、パスワードリセット機能は実装準備中です。お困りの場合は [お問い合わせ](#12-お問い合わせ) ページよりご連絡ください。

---

### Q4. メール認証のメールが届きません。

**A:** 以下をご確認ください。
1. **迷惑メールフォルダ** を確認してください
2. 入力したメールアドレスが正しいかご確認ください
3. メールは送信後 **24時間以内** に届きます
4. 上記でも解決しない場合は [お問い合わせ](#12-お問い合わせ) へご連絡ください

---

### Q5. 商品詳細が見られません。

**A:** 商品詳細ページは以下の条件を満たす方のみ閲覧できます。
- **会員登録済みでログインしている**、または
- **有田陶器市期間中（4月29日〜5月5日）** にアクセスしている

会員登録（無料）をしていただくと年間を通じてご利用いただけます。

---

### Q6. 価格情報が表示されません。

**A:** 価格情報は **会員限定** の情報です。[会員登録](#3-会員登録の手順) の後、ログインするとご確認いただけます。

---

### Q7. 会社ユーザーとして登録するにはどうすればいいですか？

**A:** 現在、会社ユーザー（COMPANY）の登録は管理者による手動設定が必要です。窯元・販売会社としてご利用を希望される方は、[お問い合わせ](#12-お問い合わせ) からご連絡ください。

---

### Q8. 商品の公開範囲「ホワイトリスト」とはどういう意味ですか？

**A:** ホワイトリストは、**特定の会員だけに商品を公開**する機能です。例えば、取引先のバイヤーだけに新商品を先行公開したい場合などにご利用いただけます。

---

### Q9. ログインセッションはどのくらい続きますか？

**A:** ログインセッションは **30日間** 有効です。30日後は再ログインが必要になります。公共のPCをご利用の場合は、使用後に必ずログアウトしてください。

---

### Q10. 対応言語に自分の言語がありません。

**A:** 現在、日本語・英語・韓国語・中国語（簡体字・繁体字）・フランス語・ドイツ語・インドネシア語の8言語に対応しています。他の言語のご要望は [お問い合わせ](#12-お問い合わせ) からお知らせください。

---

### Q11. 登録した会社情報を削除したいです。

**A:** 会社ポータルの **「会社設定」** ページから削除が可能です。ただし、削除すると登録済みの商品も合わせて削除されます。削除前に必要な情報をバックアップしてください。

---

### Q12. スマートフォンでも使えますか？

**A:** はい、レスポンシブデザイン対応ですので、スマートフォン・タブレットでもご利用いただけます。
